About Project

SAR satellites provide useful information for object detection. A methodology needs to be developed to detect ships at ocean area using SAR data and estimate size of the detected ships. The methodology should be applicable for different resolution SAR data of same bands. 

Output detected ships as a vector file, with an estimate of the size of ship.
